Rails Recipe Project
This project is meant to be an easy and convenient way to share recipes with food lovers across the world. The app is very user friendly as there is links that can navigate you to any page you would need to visit. just read the links and get on your way to making great food!

Getting Started To get started you will have to do a git clone of git@github.com:mendozacm/rails-recipe-project.git then run bundle install, run rails s, and then create an account

Prerequisites An in browser IDE or local IDE

Installing -run "git clone git@github.com:mendozacm/rails-recipe-project.git -type "cd rails-recipe-project/" into the terminal -run bundle install -run rails s -go to the generated web adress
